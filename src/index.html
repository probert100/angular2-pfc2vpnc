PCF file decoder

This is a simple user interface to pcf2vpnc and cisco-decrypt command lines.

These utilities are used to decrypt Cisco pcf vpn client group password and make PCF file more readable.

If you don't want to send your encrypted group password over the internet I recommend using this utility: http://artemkin.github.io/cisco-password-decoder/. This runs in your web browser it uses a JavaScript based implementation of the same algorithm used in the cisco-decrypt.c
